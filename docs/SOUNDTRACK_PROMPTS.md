## Music Style Guide

All prompts below are written for a consistent sonic identity across the Wizard RPG. Append the following **style suffix** to any custom prompt for cohesion:

> **Style Suffix:** `fantasy orchestral, magical atmosphere, wizard school setting, warm reverb, cinematic mix`

For chiptune or retro game variants, replace the suffix with:

> **Retro Suffix:** `8-bit chiptune, fantasy RPG, bright arpeggios, retro game soundtrack, clean mix`

### General Production Notes

- **Looping:** Most background tracks should feel seamless when looped. Avoid hard endings — prefer fade-outs or sustained final chords.
- **Duration:** Target 60–120 seconds per track for background music loops.
- **Tempo Reference:** Exploration tracks 70–90 BPM, battle tracks 130–160 BPM, ambient tracks 50–70 BPM.

**Scene:** Title screen and main menu. The player's first impression of the wizard school — wonder, grandeur, and a hint of adventure.

**Prompt:**

## Usage Notes

### Generating Music with ACE-Step 1.5

1. **Copy the prompt** from any section above into ACE-Step 1.5.
2. **Set duration** to 60–120 seconds for background loops, or 15–30 seconds for fanfares.
3. **Iterate** by changing one element at a time (e.g., swap "harp" for "piano", or adjust BPM).
4. **Export** as `.mp3` or `.wav` for web use.

### Integration with the Wizard RPG Frontend

Generated audio files can be added to `frontend/public/audio/` and loaded via the Vue application. Consider:
- Using the [Web Audio API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API) or an `<audio>` element for playback.
- Implementing a volume control in the game settings.
- Cross-fading between tracks on scene transitions.
- Looping background music seamlessly with the `loop` attribute.
